# Blueprint: Maracatu Tech Landing Page

## Overview

This document outlines the plan for creating a landing page for a technology event called "Maracatu Tech". The landing page will be a modern, visually appealing, and informative single-page website.

## Project Outline

### Style and Design

*   **Theme:** A fusion of traditional Brazilian culture (Maracatu) and modern technology.
*   **Color Palette:** Vibrant and energetic, inspired by the colors of Maracatu costumes and the Recife and Olinda carnival, mixed with a modern and techie feel. We will use a palette of bright colors like orange, blue and yellow, with a dark background to create contrast.
*   **Typography:** Expressive and modern fonts. A bold, display font for headings and a clean, readable font for body text.
*   **Imagery:** A background image that evokes the spirit of Maracatu and technology.
*   **Layout:** A single-page layout with a clear and intuitive structure. The sections will be organized in a logical flow, guiding the user through the event information.

### Features

*   **Header:** With the event logo and navigation links.
*   **Hero Section:** With a catchy headline, the event date, and a call-to-action button.
*   **About Section:** A brief description of the event.
*   **Speakers Section:** Featuring the event speakers with their photos and bios.
*   **Schedule Section:** A timeline of the event's activities.
*   **Sponsors Section:** With the logos of the event sponsors.
*   **Footer:** With contact information and social media links.
*   **Responsive Design:** The landing page will be fully responsive and accessible on all devices.

## Current Plan

This is the initial plan for the Maracatu Tech landing page. The following steps will be taken to implement it:

1.  **Create the basic HTML structure** for the landing page in `index.html`.
2.  **Style the landing page** using modern CSS in `style.css`.
3.  **Add interactivity** to the page using JavaScript in `main.js`.
4.  **Find a suitable background image** that reflects the theme of the event.
5.  **Create a logo** for the event.
